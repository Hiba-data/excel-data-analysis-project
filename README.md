# 📊 Excel Data Analysis Project – Europe Bike Store Sales

## 📌 Project Overview

This project presents a complete data analysis of the **Europe Bike Store Sales** dataset (Kaggle) using **Microsoft Excel**.

The objective is to clean, analyze, and visualize sales data in order to extract meaningful business insights and support data-driven decision-making.

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Data Cleaning
* Data Visualization (Charts & Dashboard)

---

## 📂 Dataset

* Source: Kaggle
* Dataset: *Europe Bike Store Sales*
* Format: `.xlsx`

---

## 🔍 Project Steps

### 1. Data Cleaning

* Removed duplicates
* Handled missing values
* Standardized data formats

### 2. Data Analysis

* Created Pivot Tables
* Aggregated sales by:

  * Country
  * Product Category
  * Customer Segment
  * Time (Year/Month)

### 3. Data Visualization

* Built charts (bar, line, pie)
* Designed an interactive dashboard in Excel

---

## 📊 Key Insights

* Identified top-performing countries and products
* Analyzed seasonal sales trends
* Detected high-value customer segments
* Highlighted revenue growth patterns

---

## 📈 Results

The project provides a clear overview of sales performance and helps identify opportunities for business growth through data-driven insights.
